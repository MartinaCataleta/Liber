# 📚 Liber - Social Reading Platform

**Liber** è una Single Page Application (SPA) sviluppata come progetto finale per il Corso di **Fondamenti Web 25/26** presso il Politecnico di Bari. La piattaforma permette ai lettori di gestire la propria libreria, scrivere recensioni e interagire con una community in tempo reale.

## 🛠 Stack Tecnologico

- **Frontend**: React.js, CSS3 (layout *from scratch*).
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Atlas).
- **Real-time**: Socket.io.
- **DevOps**: Docker, Docker Compose, **Deployment Cloud su Render**.

## 🌐 Live Demo & API Documentation

Puoi testare la piattaforma direttamente in produzione e consultare la documentazione tecnica aggiornata ai seguenti link:

- **Piattaforma Liber (Live Demo):** https://liber-1.onrender.com/
- **Documentazione API (Swagger/OpenAPI)** : https://liber-yogd.onrender.com/api-docs

> La documentazione Swagger è interattiva: per testare le rotte protette (come la visualizzazione della libreria o la gestione recensioni), ricordati di cliccare sul tasto **Authorize** nell'interfaccia Swagger e inserire il tuo `accessToken` ottenuto dopo il login.
